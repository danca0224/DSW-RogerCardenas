Proposito

El proposito del curso tiene como finalidad enfocar de forma asertiva los conocimientos de desarrollo
y poderlos aplicar al requerimiento del cliente.

Introducción:

Apropiar los conceptos del curso diseño de sitios web y aplicarlos mediante el desarrollo de las estructuras básicas de CSS (etiquetas, atributos y valores).

✓ Fundamentos de CSS:
CSS son las siglas en inglés para «hojas de estilo en cascada» (Cascading Style Sheets). Básicamente, es un lenguaje que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un usuario las visita. Funciona junto con el lenguaje HTML que se encarga del contenido básico de las páginas.

-Santos, D. (2022, August 8). Introducción al CSS: qué es, para qué sirve y otras 10
preguntas frecuentes. Hubspot.es. https://blog.hubspot.es/website/que-es-css

✓ Frameworks CSS:
Un framework es un esquema o marco de trabajo que ofrece una estructura base para elaborar un proyecto con objetivos específicos, una especie de plantilla que sirve como punto de partida para la organización y desarrollo de software.

-Framework. (2021, August 19). Edix España. https://www.edix.com/es/instituto/framework/
